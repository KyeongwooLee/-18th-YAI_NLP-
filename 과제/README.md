# [18th] YAI 여름방학 분기 기초NLP팀 과제 

이 저장소는 YAI 여름방학 분기 기초NLP팀 과제를 제공합니다.

PyTorch를 이용해 선형회귀부터 MLP, CNN, ResNet, RNN까지 단계적으로 학습하는 것을 목표로 합니다. 각 과제는 기본 코드 실행, 핵심 코드 작성, 모델 변형 실험, 결과 분석 순서로 구성되어 있습니다.

## 학습 목표

이 과정을 통해 다음 내용을 학습합니다.

- 머신러닝과 딥러닝의 기본 학습 과정
- PyTorch Tensor와 nn.Module 사용법
- Linear Regression과 MLP의 차이
- Loss Function과 Optimizer의 역할
- Overfitting과 Regularization
- CNN의 기본 구조
- RNN, LSTM, GRU의 기본 개념
- 실험 결과를 비교하고 해석하는 방법
  
## 과제 수행 방법
1. 해당 주차의 폴더를 엽니다.
2. assignment.ipynb 파일을 Google Colab 또는 Jupyter Notebook에서 실행합니다.
3. TODO와 빈칸을 모두 완성합니다.
4. 코드 셀을 위에서부터 순서대로 실행합니다.
5. 모든 셀이 정상적으로 실행되며 요구 수치를 달성하면 완료입니다.
